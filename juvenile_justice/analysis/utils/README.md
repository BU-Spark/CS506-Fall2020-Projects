## springfield_create_csv.py
Download all existing arrest logs in PDF form from the Springfield police department's website and output a CSV containing information on each arrest
This is included to show the process of how we created the springfield_arrest_logs.csv dataset

# springfield_merge_w_nibrs.py
Merge the existing springfield_arrest_logs.csv dataset with the nibrs_2017 and nibrs_2018 CSVs in order to get race and ethnicity information on the suspects described in the Springfield arrest logs
